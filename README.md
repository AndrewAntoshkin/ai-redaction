# ai-redaction

Лендинг продукта **«ИИ-редакция для Telegram-медиа»** — конфигурируемой платформы под бриф клиента (источники, темы, форматы и тон собираются под бренд).

## Открыть онлайн

После включения GitHub Pages: https://andrewantoshkin.github.io/ai-redaction/

## Локально

```bash
python3 -m http.server 8765
```

Открыть в браузере: http://127.0.0.1:8765/

## Структура

- `index.html` — лендинг (одностраничник, всё инлайн: HTML + CSS, без сборки)
- `assets/brief-editorial-ai.docx` — бриф для скачивания
